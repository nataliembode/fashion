How to get your Instagram API token

1. Go to developers.facebook.com → Create App → choose "Business" type

2. Add "Facebook Login for Business" product → save

3. Tools → Graph API Explorer → select your app → add permissions: instagram_basic, instagram_manage_insights → Generate Token

4. Extend it: developers.facebook.com/tools/debug/accesstoken → "Extend Access Token" (60 days)
